프로그래밍 실습하기

🎯 프로그래밍 과정을 알아보자

💡 학습 팁
  • 플립북 실행버튼을 클릭해 주세요.
  • 주제별 블록코딩 결과를 확인해 보세요.

🎯 묻고 대답하기

💡 학습팁
  • 녹색깃발을 클릭해 주세요.
  • 입력과 출력을 연습해 보세요.

🎯 자동차 미로찾기

💡 학습팁
  • 녹색깃발을 클릭해 주세요.
  • 방향키를 눌러 자동차를 깃발까지 이동해 보세요.
  • 10초 안에 깃발까지 도착해야 합니다.
  • 미로 선에 닿으면 원위치로 돌아갑니다.

🎯 병아리 미로찾기

💡 학습팁
  • 녹색깃발을 클릭해 주세요.
  • 방향키를 눌러 미로 따라 엄마 닭으로 이동해 보세요.
  • 60초 안에 엄마 닭 못 찾으면 게임오버입니다.
  • 뱀과 미로 선에 닿으면 원위치로 돌아갑니다.

🎯 음료수 자동판매기

💡 학습팁
  • 녹색깃발을 클릭해 주세요.
  • 음료수를 선택해 주세요.
  • 오른쪽 리스트에 선택한 음료의 개수가 표시됩니다.
  • 계산 버튼을 클릭하신 후 거스름돈을 받거나 부족한 금액을 더 내 주세요.

온라인 실습 플랫폼

  • CodeUp - 한국어 코딩테스트/실습
  • HackerRank - 다양한 언어와 난이도의 실습 문제
  • LeetCode - 인터뷰 대비 실습 문제
  • 엔트리 - 블록형 코딩 학습 플랫폼 (국내)
  • 스크래치 - 블록형 코딩 학습 플랫폼 (해외)

온라인 실습 UI

  • JSFiddle - HTML/CSS/JS 실습, 로그인 없이 사용 가능, 임베드 지원 바로가기
  • JDoodle - 70개 이상의 언어 지원, 로그인 없이 사용 가능, 임베드 지원 바로가기
  • Replit - 다양한 언어 지원, 로그인 없이 일부 언어 실습 가능, 임베드 지원 바로가기
  • CodePen - 웹 프론트엔드 실습, 로그인 없이 사용 가능, 임베드 지원 바로가기
